Clear Chat Function — 100 Test Questions
=========================================
Version 1.0 | 2026-04-15


How to use
----------
Each question simulates a real user message sent to the /chat endpoint.

For each question:
  - Expected tool: Which tool the LLM should call (or "None" if no tool is needed)
  - Expected args: Key arguments the tool should be called with
  - Notes: Additional context about the test case

Questions marked as "follow-up" (section 11) should be sent as a second message
in a conversation after a relevant first question.
